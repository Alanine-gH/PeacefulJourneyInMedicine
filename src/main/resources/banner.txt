✨🎯 项目名称：医路安心——AI 驱动的一站式跨境就医服务平台

🌟 核心技术矩阵
▪️ NLP 智能分诊 + 医学知识图谱构建
▪️ Transformer 多语言机器翻译（支持 20+ 语言）
▪️ LBS 智能调度引擎 + 实时位置追踪
▪️ OCR 病历识别 + 处方解析技术
▪️ 双层加密（AES+PIPL）医疗数据安全防护
🎯 核心功能亮点
✅ 多语言 AI 智能分诊，3-5 分钟生成就医建议书
✅ 7×24 小时客服代预约，对接三甲医院国际部
✅ 专业陪诊师全程陪同，实时双语翻译
✅ 落地接待 + 送机服务，全程中文就医无忧
✅ 跨境支付网关（Stripe/Adyen/Alipay/WeChat Pay）
✅ 诊后增值服务：病历翻译、处方解读、远程随访
🛠️ 技术框架
Spring Boot Version: ${spring-boot.version}
前端：Vue.js + Uni-app（跨平台兼容）
后端：Spring Boot + MyBatis-Plus
缓存数据库：MySQL + MongoDB + MinIO + Redis
AI 引擎：Python + TensorFlow + PyTorch
